# Controle de Corridas

Este projeto simula o controle de corridas de um entregador, com o objetivo de calcular o total ganho, a média das corridas, verificar se a meta do dia foi batida, e salvar os resultados em um arquivo de texto.

## Funcionalidades

- Inserir valores das corridas.
- Calcular total, média, maior e menor corrida.
- Verificar se a meta foi batida.
- Salvar os resultados em um arquivo de texto.
